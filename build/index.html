Тестовое задание: Frontend разработчик

Пункт 1: реализовать баннер

Баннер должен перелистываться. Для реализации механизма перелистывания запрещено использовать готовые JS плагины, должен быть написан свой код слайдинга на JavaSсript либо jQuery.

Результат

1. обычный не зацикленный слайдер

2. зацикленная версия

3. зацикленный с автовоспроизведением и остановкой при наведении
